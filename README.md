# gogo

## build a exe file

    C:\..\main>go build -o gogo.exe

## run the execute file

    C:\..\main>gogo.exe
    C:\..\main>gogo.exe try
    C:\..\main>gogo.exe try g
    C:\..\main>gogo.exe 1
