# gogo

## build a exe file

    C:\..\main>go build -o gogo.exe

## run the execute file

Please try different parameters to see what the results are.

    C:\..\main>gogo.exe 
    C:\..\main>gogo.exe test
    C:\..\main>gogo.exe run
    C:\..\main>gogo.exe others
